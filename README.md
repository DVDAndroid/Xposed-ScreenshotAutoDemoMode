Screenshot Auto Demo Mode - Xposed
==================================
[![Build Status](https://travis-ci.org/DVDAndroid/Xposed-ScreenshotAutoDemoMode.svg?branch=material)](https://travis-ci.org/DVDAndroid/Xposed-ScreenshotAutoDemoMode)
